Largest Sum Contiguous Subarray OR Kadane's Algorithm

Write an efficient program to find the sum of contiguous subarray within a one-dimensional array of numbers which has the largest sum.

EXPLANATION:

We can easily solve this in linear time using kadane's algorithm. The idea is to maintain maximum (positive sum) sub-array "ending" at each index of the given array. This subarray is either empty(in which case its sum is zero) or consists of one more element than the maximum subarray ending at the previous index.

    Input:{-2,1,-3,4,-1,2,1,-5,4}
   Output:Subarray with largest sum is {4,-1,2,1} with sum 6

JAVA CODE:

           	class Main
{
    // Function to find contiguous sub-array with the largest sum
	// in given set of integers (handles negative numbers as well)
	public static int kadane(int[] A)
	{
		// stores maximum sum sub-array found so far
		int maxSoFar = Integer.MIN_VALUE;

		// stores maximum sum of sub-array ending at current position
		int maxEndingHere = 0;

		// traverse the given array
		for (int i: A)
		{
			// update maximum sum of sub-array "ending" at index i (by adding
			// current element to maximum sum ending at previous index)
			maxEndingHere = maxEndingHere + i;

			// maximum sum is should be more than the current element
			maxEndingHere = Integer.max(maxEndingHere, i);

			// update result if current sub-array sum is found to be greater
			maxSoFar = Integer.max(maxSoFar, maxEndingHere);
		}

		return maxSoFar;
	}

	public static void main(String[] args)
	{
		int[] A = { -8, -3, -6, -2, -5, -4 };

		System.out.println("The sum of contiguous subarray with the largest sum is " +
				kadaneNeg(A));

	}
}

N/2 Greatest Number OR Moore's Voting Algorithm

Write a program to find which number in the given array is occuring more than N/2 times, where N is number of elements in the array, if it given that one such number always exist.

EXPLANATION:

    This is a two-step process: 
         (1)The first step gives the element that maybe the majority element in the array. If there is 
         a majority element in an array, then this step will definitely return majority element, otherwise, 
         it will return candidate for majority element.
         (2)Check if the element obtained from the above step is majority element. This step is necessary 
         as there might be no majority element.
 	
 		Input:{ 1, 3, 3, 3, 2 }
 		Output:3

JAVA CODE:

  class Solution {
    public int majorityElement(int[] nums) {
        // using Moore's Voting Algorithm
        int maj_index=0, count = 1;
        int i;
        for(i=0;i<nums.length;i++){
            if(nums[maj_index]==nums[i]){
                count++;
            }
            else{
                count--;
            }
            if(count==0){
                maj_index=i;
                count=1;
            }
        }
        return nums[maj_index];
        
    }
}

N/3 Greatest Number

Given an integer array of size n, find all elements that appear more than n/3 times.

        Input:[3,2,3]
 		Output:3
 		Input:[1,2]
 		Output:[1,2]

JAVA CODE:

  class Solution {
    public List<Integer> majorityElement(int[] nums) {
      List<Integer> ans = new ArrayList<>();
      int n=nums.length;  
      DataStructure[] ds = new DataStructure[2];
      ds[0] = new DataStructure();
      ds[1] = new DataStructure(); 
      for(int i=0;i<n;i++)
      {
          int j;
          inner: for(j=0;j<2;j++)
          {
              if(ds[j].e==nums[i])
              {
                  ds[j].c +=1;
                  break inner;
              }
          }
          /* If nums[i] is not present in ds[] */
        if (j == 2) 
        { 
            int l; 
              
            /* If there is position available in ds[], then place  
              nums[i] in the first available position and set count as 1*/
            inner: for (l=0; l<2; l++) 
            { 
                if (ds[l].c == 0) 
                { 
                    ds[l].e = nums[i]; 
                    ds[l].c = 1; 
                    break inner; 
                } 
            } 
  
            /* If all the position in the ds[] are filled, then  
               decrease count of every element by 1 */
            if (l == 2) 
                for (l=0; l<2; l++) 
                    ds[l].c -= 1; 
        } 
    }
        /*Step 3: Check actual counts of potential candidates in ds[]*/
        int lim= ds[1].c>0?2:1;
    for (int i=0; i<lim; i++) 
    { 
        // Calculate actual count of elements
        int ac = 0;  // actual count 
        for (int j=0; j<n; j++) 
            if (nums[j] == ds[i].e) 
                ac++; 
  
        // If actual count is more than n/k, then print it
        if (ac > n/3) 
           ans.add(ds[i].e); 
    }
     return ans;
    }
}
class DataStructure
{
    int e;
    int c;
    public void DataStructure(){
        e=0;
        c=0;
    }
}

Merge Overlapping Intervals

Given a collection of intervals, merge all overlapping intervals.

        Input:[[1,3],[2,6],[8,10],[15,18]]
 		Output:[[1,6],[8,10],[15,18]]
 		Explanation: Since intervals [1,3] and [2,6] overlaps, merge them into [1,6].
 		Input:[[1,4],[4,5]]
 		Output:[[1,5]]
 		Explanation: Intervals [1,4] and [4,5] are considered overlapping.

JAVA CODE:

  class Solution {
    public int[][] merge(int[][] intervals) {
        int n=intervals.length;
        List<Job> ans = new ArrayList<>();
        for(int i=0;i<n;i++)
        {
          ans.add(new Job((intervals[i][0]), (intervals[i][1])));
        }
        Collections.sort(ans , Comparator.comparingInt( x -> x.start));
        for(int i=0;i<n-1;) {
        	if(ans.get(i).finish>=ans.get(i+1).start) {
        		ans.get(i).finish = Math.max(ans.get(i+1).finish, 
        		                             ans.get(i).finish);
        		ans.remove(i+1);
        		n--;
        	    continue;
        	}
        	i++;
        }
        int[][] temp = new int[ans.size()][2];
        for(int i=0;i<ans.size();i++) {
        	temp[i][0]=ans.get(i).start;
        	temp[i][1]=ans.get(i).finish;
        }
        return temp;
        }
}
class Job{
    int start,finish;
    Job( int start , int finish){
        this.start = start;
        this.finish = finish;
    }
}

Rotate Matrix

You are given an n x n 2D matrix representing an image, rotate the image by 90 degrees (clockwise).

        Input:matrix = [[1,2,3],[4,5,6],[7,8,9]]
 		Output:[[7,4,1],[8,5,2],[9,6,3]]
 		Input:matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]
 		Output:[[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]

JAVA CODE:

  class Solution {
    public void rotate(int[][] matrix) {
       int n=matrix.length;
       int x1=0,y1=0,x2=0,y2=0,x3=0,y3=0,x4=0,y4=0;
        int k=n;
       for(int i=1;i<=n/2;i++){
           x1=i-1;
           y1=i-1;
           x2=i-1;
           y2=n-i;
           x3=n-i;
           y3=n-i;
           x4=n-i;
           y4=i-1;
           for(int j=1;j<k;j++){
           int temp=matrix[x4][y4];
           matrix[x4][y4] = matrix[x3][y3];
           matrix[x3][y3] = matrix[x2][y2];
           matrix[x2][y2] = matrix[x1][y1];
           matrix[x1][y1] = temp;
           y1++;
           x2++;
           y3--;
           x4--;
           }
           k=k-2;
       }
    }
}

If you were only permitted to complete at most one transaction (i.e., buy one and sell one share of the stock), design an algorithm to find the maximum profit.
Note that you cannot sell a stock before you buy one. 

        Input:[7,1,5,3,6,4]
 		Output:5
 		Input:matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]
 		Output:[[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]
